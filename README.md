# ESP32 Autonomous Surveillance & Bomb Detection Rover

An ESP32-CAM based autonomous surveillance rover capable of:

- Live video streaming over WiFi
- Remote motor and pan-tilt control
- Multi-sensor bomb detection (Gas + Metal + Temperature)
- Real-time telemetry monitoring
- Modular embedded firmware architecture

## Hardware

- ESP32-CAM
- L298N Motor Driver
- MQ Gas Sensor
- Metal Detector Module
- DHT11/DHT22 Temperature Sensor
- Pan-Tilt Servo
- 4WD Chassis

## Features

- Async WebSocket control
- Real-time JPEG streaming
- Sensor fusion based threat detection
- PWM motor speed control
- PSRAM optimized camera pipeline
